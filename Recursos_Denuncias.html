Recursos y Denuncias

Certificaciones en Ciberseguridad

  • CERT México(Computer Emergency Response Team): Ofrece orientación sobre respuesta a incidentes mexicanos.
  • CERT España(Computer Emergency Response Team): Ofrece orientación sobre respuesta a incidentes españoles.
  • CERT E.U. (Computer Emergency Response Team): Ofrece orientación sobre respuesta a incidentes estadounidenses.
  • OWASP (Open Web Application Security Project): Enfocado en la seguridad de aplicaciones web.
  • CISA (Cybersecurity & Infrastructure Security Agency): Información sobre amenazas y protección cibernética.

Cómo Reportar Incidentes de Ciberseguridad

Si eres víctima de un ataque cibernético o sospechas de actividad maliciosa, sigue estos pasos:

  1. Contacta a tu proveedor de servicios de Internet (ISP).
  2. Informa a la policía local o a las autoridades pertinentes.
  3. Utiliza plataformas de denuncia en línea para reportar fraudes y estafas.
  4. Preserva cualquier evidencia, como correos electrónicos, capturas de pantalla, etc.

Otros Recursos Útiles

  • Certified Information Systems Security Professional (CISSP): Certificación avanzada para profesionales de seguridad.
  • Certified Ethical Hacker (CEH): Para quienes buscan entender y abordar los problemas de seguridad desde la perspectiva de un hacker.
  • CompTIA Security+: Certificación básica en seguridad informática.
  • Certified Information Security Manager (CISM): Orientada a la gestión y gestión de riesgos.
  • Certified Information Systems Auditor (CISA): Para auditores de sistemas de información.
