Amenazas Comunes para los Usuarios

Ingeniería Social

La ingeniería social manipula a las personas para que revelen información confidencial o realicen acciones inseguras. Esta técnica utiliza la confianza para explotar las vulnerabilidades humanas, a menudo mediante llamadas o mensajes falsos.

Phishing

El phishing engaña a los usuarios para que compartan datos confidenciales, como contraseñas, mediante correos electrónicos o mensajes que parecen ser de fuentes confiables.

Drive-By Downloads

Estas descargas ocurren cuando se visita un sitio web comprometido que instala software malicioso sin el conocimiento o consentimiento del usuario.

Rootkit

Los rootkits son herramientas que permiten a un atacante mantener el control de un sistema sin ser detectado, ocultando su presencia en el sistema comprometido.

Puertas Traseras

Las puertas traseras permiten el acceso no autorizado a un sistema, dejando un “camino oculto” para que los atacantes accedan a la información en cualquier momento.

Botnet

Las botnets son redes de dispositivos infectados que los atacantes controlan remotamente para realizar ataques coordinados, como enviar spam o realizar ataques DDoS.

Keylogger

Un keylogger registra las pulsaciones del teclado del usuario, permitiendo a los atacantes obtener información sensible, como contraseñas o datos bancarios.

Ransomware

El ransomware encripta los archivos de un usuario y exige un rescate para desbloquearlos. Esta amenaza ha afectado tanto a individuos como a organizaciones.

Adware

El adware muestra anuncios no deseados en el dispositivo del usuario, a menudo recopilando información sobre sus hábitos de navegación para mostrar anuncios personalizados.

Spyware

El spyware monitorea y recopila información sobre las actividades del usuario sin su conocimiento, a menudo con fines de espionaje o robo de datos personales.
